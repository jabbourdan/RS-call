# Wiki Guidelines — Voicely CRM

> **Rule:** Always read every file in this folder before creating or updating any Confluence wiki page for the Voicely CRM project.

This folder defines the structure, tone, and content requirements for the project's Confluence wiki under the **Jabbour Test wiki (JA)** space on `wearabledevices-ailab.atlassian.net`.

## Files

| File | Purpose |
|---|---|
| `wiki-structure.md` | Page hierarchy and what each page must cover |
| `style-guide.md` | Writing tone, formatting rules, and conventions |
| `content-sources.md` | Where to pull information from in the codebase |
